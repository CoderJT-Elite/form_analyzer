# Form Analyzer

This repository contains:
- `app/`: the Flutter mobile app (primary product)
- `web/`: landing page assets

## Current Product Status (MVP)

The MVP target is a stable, testable mobile app for real-time squat form feedback. Marketing/site assets are kept lightweight and secondary to app reliability.

For app setup, architecture, and validation steps, use:
- [`app/README.md`](app/README.md)

## Quick Start

```bash
cd app
flutter pub get
flutter run
```

## MVP Review Checklist

- Keep one clear core use case (squat analysis)
- Prefer maintainability over feature expansion
- Keep docs and commands consistent
- Ensure tests/analyze are runnable in a standard Flutter environment
